# CCDataLink

#### 介绍

CC数据库连接工具 基于eletron + vue + node + ts + java

v1.0.2 预览版已发布 （没买代码签名，安装报毒请忽略）

优化了列表卡顿

包含 数据库连接 数据库sql执行 关键词提示等

目前支持mysql MariaDB oracle sqlserver

其中oracle不支持多条sql同时执行
